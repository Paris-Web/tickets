# Retours et remarques sur les sites de Paris Web / 24 jours de web

Ce dépôt accueille et centralisera les retours et suggestions sur les sites de Paris Web et 24 jours de web.

- <https://www.paris-web.fr>
- <https://appel.paris-web.fr>
- <https://inscriptions.paris-web.fr>

## Participer

Toute personne de bonne volonté est invitée à partager ses remarques pour améliorer la qualité, l’accessibilité, les contenus ou autre, des sites de Paris Web listé ci-dessus.

L’équipe de Paris Web répondra aux tickets dans la mesure du possible, sans garantie que les changements proposés seront réalisés.
Le [Code de Conduite](https://www.paris-web.fr/code-de-conduite) s’applique aux discussions.
